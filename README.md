# Feel Better - مشروع دعم نفسي مبسط

## الفكرة (Problem - Solution)
- **المشكلة:** بعد الحرب، كثير من الناس بيعانوا من ضغط نفسي وبيحتاجوا مساحة بسيطة للدعم.
- **الحل:** موقع إلكتروني بسيط يعرض نصائح يومية، فيديو دعم نفسي، ومعرض صور مهدئ، مع إمكانية التواصل.

## الفئة المستهدفة (Target Users)
- الأفراد في المجتمع اللي بيبحثوا عن دعم نفسي بسيط.
- أي شخص محتاج نصائح يومية أو محتوى يساعده على الاسترخاء.

## أهم الميزات (Features)
- جدول نصائح يومية منسق.
- فيديو دعم نفسي يمكن تشغيله مباشرة.
- معرض صور بسيط (3 صور).
- نموذج تواصل لإرسال رسالة.
- زر الوضع الليلي لتغيير شكل الموقع.

## التقنيات المستخدمة (Technologies)
- **HTML5**: لبناء هيكل الصفحة.
- **CSS3 + Bootstrap**: لتنسيق الأزرار، الجداول، الصور، والمعرض.
- **JavaScript + jQuery**: لإضافة التفاعل (الوضع الليلي، رسالة تأكيد عند إرسال النموذج).
- **Font Awesome**: لإضافة أيقونات معبرة.
